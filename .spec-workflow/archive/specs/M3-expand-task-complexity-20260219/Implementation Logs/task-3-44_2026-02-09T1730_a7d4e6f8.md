### Components

#### ApprovalPanel (new)
- **Location:** client/src/components/ApprovalPanel.tsx
- **Purpose:** Reusable approval/revision/rejection panel for both quest-level and task-level approvals
- **Props:** `entityType: 'quest' | 'task'`, `onSubmit: (decision, feedback?) => Promise<void>`, `hidden?: boolean`
- **Features:**
  - Radio button decision selector (Approve / Request Revision / Reject)
  - Expandable comment textarea (required for revision/rejection)
  - Rejection confirmation modal dialog
  - Loading state with spinner
  - Validation error display
  - Color-coded submit button per decision type

| Method | Route | Purpose |
|--------|-------|---------|
| `approveTask(taskId, feedback?)` | POST /api/tasks/:taskId/approve | Approve task via agent-producer |
| `reviseTask(taskId, feedback)` | POST /api/tasks/:taskId/revise | Request task revision |
| `rejectTask(taskId, feedback)` | POST /api/tasks/:taskId/reject | Reject task |
